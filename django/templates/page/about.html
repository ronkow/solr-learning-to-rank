About this project

The primary goal of this project is to develop a grammar question recommender system driven by machine learning. By providing an input sentence, we wish to search for sentences in our question bank with similar grammatical structures and rank the results. This grammar learning website is designed to facilitate that. The following are the features of this website.

Grammar learning

  • Multiple-choice quizzes for a list of grammar topics
  • Random order of questions and answer choices in each quiz
  • User's quiz results displayed on submission of each quiz
  • For wrong answers submitted, correct answers are displayed for logged in users

Question search

  • Basic search (by Django) of questions from a question bank, by text string matching or by topic
  • Search engine by Apache Solr using BM25 similarity scoring
  • Intelligent search by Apache Solr with Learning to Rank (development in progress)

Security

  • Member registration (signup)
  • User authentication (login)
  • reCAPTCHA check for login and signup
  • Password change and reset for members
